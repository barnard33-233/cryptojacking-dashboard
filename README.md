## How to run in developing environment

### frontend

Enter directory dashboard/front/

1. install dependencies
```shell
npm install
```

2. build
```shell
npm run build
```

### server

Enter directory dashboard/

1. migration:
```shell
python manage.py migration
```

2. run server
```shell
python manage.py runserver --noreload
```
